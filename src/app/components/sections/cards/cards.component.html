CleanArchMvc

O CleanArchMvc é uma aplicação desenvolvida em C# utilizando ASP.NET Core que implementa os princípios da Clean Architecture e do SOLID com a arquitetura MVC (Model-View-Controller). O projeto contém uma versão visual do site, permitindo gerenciar categorias e produtos, além de uma API REST segura para operações externas com a utilização do token jwt.
